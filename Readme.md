# Darkboy1
## Description
A simple JavaFX project demonstrating GUI application concepts.
## Requirements
- Java 21 or higher
- JavaFX SDK 21 or higher
## How to Run (PowerShell)
1. Clone this repository.
2. Open PowerShell in project folder.
3. Compile:
```powershell
javac --module-path "C:\path\to\javafx-sdk-21\lib" --add-modules javafx.controls,javafx.fxml Darkboy1.java
